🚀 Démarrer un serveur local (Simple)
Ouvrez un terminal dans ce dossier et exécutez une de ces commandes :
Python (recommandé) :
python3 -m http.server 8080
Node.js :
npx http-server -p 8080
PHP :
php -S localhost:8080

Puis ouvrez : http://localhost:8080
