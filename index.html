📡 Miroir d'écran natif Android Android dispose d'une fonction Cast/Miracast intégrée. Le receiver peut regarder sur une TV, un PC ou via ce site.
1

Ouvrir les Paramètres rapides

Glissez vers le bas deux fois pour afficher tous les raccourcis. Cherchez Cast, Diffusion, Smart View (Samsung) ou Partage d'écran.

2

Choisir la cible

Sélectionnez votre Chromecast, TV Android, ou un PC avec LetsView / AirDroid Cast installé.

3

Receiver regarde sur le PC

Une fois l'écran casté sur le PC, ouvrez ce site sur le PC et partagez la capture d'écran du PC vers le receiver — connexion en 2 étapes.

📲 Apps gratuites "Cast vers PC"

  • LetsView — Android → PC, gratuit, Miracast/AirPlay
  • AirDroid Cast — Android → PC via navigateur ou app
  • ApowerMirror — Android → PC, WiFi ou USB
🌐 LetsView 🌐 AirDroid Cast 🌐 ApowerMirror
🔌 scrcpy — Miroir USB gratuit et open source La solution la plus fiable pour mirror Android → PC. Aucune latence, HD, gratuit.
1

Activer le débogage USB

Paramètres → À propos du téléphone → tapez 7x sur Numéro de build → Options développeur → Débogage USB : activé.

2

Installer scrcpy sur le PC

Téléchargez sur GitHub : scrcpy (Windows/Mac/Linux). Branchez l'Android avec un câble USB. Lancez scrcpy dans un terminal.

3

Partager depuis le PC

L'écran Android apparaît dans une fenêtre sur le PC. Utilisez ensuite ce site depuis le PC pour envoyer ce flux au receiver.

📥 Télécharger scrcpy
📲 Applications Android avec partage d'écran intégré Ces apps contournent la limitation du navigateur en utilisant les permissions système Android.

Applications recommandées

  1. TeamViewer QuickSupport — partage d'écran Android vers n'importe quel appareil
  2. AnyDesk — accès à distance et miroir d'écran
  3. Vysor — miroir Android → Chrome (extension PC)
  4. Google Meet / Zoom — partage d'écran Android intégré dans les appels
  5. Deskreen — miroir PC → tout appareil avec navigateur
📱 TeamViewer 📱 AnyDesk 📱 Vysor
Pourquoi le navigateur ne peut pas capturer l'écran Android ? Google a délibérément désactivé getDisplayMedia dans Chrome Android depuis Chrome 88. Ce n'est pas un bug — c'est une décision de sécurité. Seules les apps natives peuvent accéder à la capture d'écran via l'API MediaProjection d'Android.

✓

iOS → caméra partagée
